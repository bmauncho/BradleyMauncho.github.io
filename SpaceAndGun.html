Space and Gun

Duration : 3 months
Platform : Blender and Unity
Role : Game developer, sound designer and UI / UX Design.

WELCOME TO SPACE AND GUN!

I was assigned to create a two-level game for students in grades 6 and under. The game should include the following elements/features: 3D animated character /characters,3D environment design, barriers and collisions, points/lives count, two levels, game start user interfaces (UI) user experience (UX) design, game end user interfaces (UI) user experience (UX) design,and congratulatory user interfaces (UI) user experience (UX) design.

CONCEPT

The player (interaction) to help overcome (obstacle) to achieve (goal). The player controls the direction of a man running around an area and avoid being killed by enemies to achieve visiting more locations and collecting objects (weapons, health and bullets) to win the game and protect the human race from the evil menace.

ENVIRONMENT DESIGN

The environment will be a simple 3d space in which the characters will be accommodated.
The environment includes a road, trees, fires and walls etc.
The game also includes objects, items and weapons as part of the environment and can be collected as the game continues.

GAME CHARACTERISTICS

It is a first person shooter(FPS) game.
The character shooting other characters(enemies).
The game has opponents(enemies).
The game character has lives(100 max).
The game has obstacles.
The game has objects, tools and weapons.
Game allows shooting opponents and other strategies to win.
Smashing obstacles is possible but limited to certain weapons.
